Formatting help

Episodes on Fiction Branches are formatted using Markdown.
HTML tags will be escaped and presented as entered in the episode (and thus should not be used).
Fiction Branches implements most basic features of Markdown (specifically the CommonMark specification). Some Markdown features (such as HTML blocks) may not be enabled.

You can create headers like:

# Header 1
## Header 2
### Header 3
#### Header 4

They will output like:

Header 1

Header 2

Header 3

Header 4



You can also make a header by underlining the text with hyphens:

This is a header
-----

becomes:

This is a header



To insert a horizontal rule, use an empty line followed by a line of hyphens:

This is not a header

-----

becomes:

This is not a header

Text surrounded by single *asterisks* or _underscores_ will become italicized.

Text surrounded by double **asterisks** or __underscores__ will become bold.

Text surrounded by -> and <- will become centered, eg ->This text is centered<- becomes:

This text is centered


To insert a link:
[This is a link](https://google.com) becomes This is a link.


To embed an image in the episode:
![alt text is optional](https://test.fictionbranches.net/images/fblogo.png) becomes:



Table example

Colons can be used to align columns.

There must be at least 3 dashes separating each header cell.

The outer pipes (|) are optional, and you don't need to make the raw Markdown line up prettily.

You can also use inline Markdown.



| Tables        | Are           | Cool  |
| ------------- |:-------------:| -----:|
| col 3 is      | right-aligned | $1600 |
| col 2 is      | centered      |   $12 |
| zebra stripes | are neat      |    $1 |

Markdown | Less | Pretty
--- | --- | ---
*Still* | `renders` | **nicely**
1 | 2 | 3
Tables Are Cool
col 3 is right-aligned $1600
col 2 is centered $12
zebra stripes are neat $1
Markdown Less Pretty
Still renders nicely
1 2 3
